O que é HTML?

HTML é a sigla de HyperText Markup Language, que em português significa "Linguagem de Marcação de Hipertexto". Consiste em uma linguagem de marcação utilizada para desenvolvimento de páginas web, por meio da criação de documentos que podem ser lidos por um interpretador, ou navegador, e transmitidos por meio da internet. Para criar documentos HTML não é necessário mais do que um editor de texto simples, embora recomende-se utilizar aplicativos especificos para este fim, que possuem recursos para facilitar a escrita dos códigos. As tags, servem para indicar a função de cada elemento da página Web e funcionam como comandos de formatação de textos, tabelas, links, imagens, entre outros. Muitas informações sobre a estrutura desta linguagem de marcação, podem ser acessadas por meio deste link nas páginas da W3Schools.

Tags e Atributos:

Em um documento HTML básico iremos encontrar tags, e dentro destas, poderemos encontrar atributos que ajudam o desenvolvedor a configurar e construir os documentos HTML. Tags também podem conter atributos, que são pedaços extra de informação. Atributos aparecem dentro da tag de abertura e seu valor está sempre dentro de aspas duplas. Até o momento vimos as tags "p", "div", "h1", "b", "em" e "span". Veremos exemplos de tags e atributos nas próximas lições. Para mais informações, podemos recorrer a documentação: HTML Attributes.

Conceitos de Span e Div:

A diferença entre Span e Div é que span é um elemento inline e geralmente é usado para um pequeno bloco de HTML, e div é um elemento de bloco (o que basicamente é equivalente a ter uma quebra de linha antes e depois do elemento).
